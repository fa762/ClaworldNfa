## 画风规范

| 项目 | 规格 |
|------|------|
| 构图 | 上半身肖像（胸部以上），正面或微侧，与 BAYC 相同取景角度 |
| 画风 | 粗黑描边、色块平涂、卡通化比例、无渐变阴影、表情有个性 |
| 龙虾 | 蹲/站在主人肩膀上，赛博机械改造风格，每只造型独特 |
| 背景 | 纯色一块，靠颜色区分稀有度 |
| 尺寸 | 1024 × 1024，1:1 方形 |
| 线条 | 粗黑轮廓线，干净利落 |
| 色彩 | 高饱和、高对比、色块分明、不用渐变 |
