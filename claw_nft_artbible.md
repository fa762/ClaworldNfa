## 3. 构图规范

### 统一构图
```
所有NFT：
  格式：1:1 正方形
  主体：人物上半身（腰部以上）
  视角：正面或3/4侧面
  龙虾：趴在人物肩膀上（特殊角色除外，见备注）
  背景：据点特征场景（模糊虚化）
```

### 特殊龙虾位置备注
| 角色 | 龙虾位置 | 原因 |
|------|----------|------|
| 楚门 + 问号 | 手心（掌上托着） | 体现珍视，问号极小 |
| Dr. Null + 奇点 | 肩侧悬浮 | 奇点已进化，不趴着 |
| Seed + 课本 | 头顶悬浮 | 课本是漂浮的书型龙虾 |
| Forge + 计时 | 焊在左前臂 | 计时是工头，不是伙伴 |
| ZERO | 无人类，纯龙虾 | 神话形态 |

| 据点 | 背景元素 |
|------|----------|
| SHELTER-01 | 地下走廊，17度蓝光，水培植物 |

### 神话 Mythic
```
人物：无（纯龙虾形态）
背景：纯白圣光 + 区块链节点粒子
光效：全图白色神光溢出
龙虾：UNIT-0000000001，白色眼睛，纯净无杂质
特效：意识碎片在周围飘散（代表挂在区块链上的意识）
边框：极细白色发光线
```

### 传说 Legendary
```
人物：精致全彩，细节丰富
背景：标志性场景，有景深
光效：强烈光晕 + 金色粒子飘散
龙虾：有进化特征，体型偏大，细节突出
边框：金色流光边框，四角有徽章装饰
```

### 史诗 Epic
```
人物：全彩，有标志性装备/道具
背景：中等景深，据点特征明显
光效：蓝紫双色渐变光晕
龙虾：明显特征，贴合性格
边框：蓝紫色发光边框
```

### 稀有 Rare
```
人物：全彩，服装有据点/职业特征
背景：简单场景
光效：单色柔和蓝色光晕
龙虾：标准体型，有性格细节
边框：蓝色细边框
```

### 普通 Common
```
人物：全彩，标准居民服装
背景：纯色/简单渐变
光效：无特效
龙虾：标准款，随机颜色
边框：灰色细边框
```
