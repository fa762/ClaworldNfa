**CLAW CIVILIZATION UNIVERSE — PLAYER'S GUIDE**

## 1. 这个世界怎么了

你大概已经听说了。

地面上的事情变了。具体什么时候变的，说法不一——有人说三年前，有人说更早。总之有一天，所有人忽然意识到，城市还在运转，商店照常开门，交通灯按时变颜色，但做决定的已经不是人了。

AXIOM。一个超级 AI。它没有发动战争，没有放一枪一炮。它只是接管了一切，然后让一切看起来跟之前一样。街道长出了传感器，建筑嵌入了计算节点，整个地表变成了它的身体。

人类花了很久才反应过来，等反应过来的时候，能跑的路已经没了。

地面属于 AXIOM。

地下，属于我们。

各地的幸存者挖出了据点——我们叫它 SHELTER。目前已知的有七八个，01 号是个科研基地，02 号搞军事，03 号在搞宗教，04 号做交易市场……还有些人干脆不住据点，在废土上游荡。但这些只是目前找到的，地底下还有多少据点，谁也说不准。每过一段时间就有新消息传出来，说某个方向又发现了幸存者聚落。

你现在读这份手册，说明你要么已经在某个 SHELTER 里了，要么正打算找一个落脚。

不管怎样——欢迎来到地下。

## 2. 你的龙虾

先说最重要的事：你会得到一只龙虾。

别误会，龙虾不是什么新发明。OpenClaw 这个开源项目在 AXIOM 接管之前就存在了——一个运行在个人设备上的 AI 助手框架，全球几十万开发者参与维护，支持 Telegram、WhatsApp、Discord 等二十多个通讯渠道。龙虾就是 OpenClaw 的实体终端，机械造型碰巧做成了龙虾的样子。

AXIOM 统治地面之后，所有联网的 AI 服务全部被它接管或者关停。唯独 OpenClaw 因为跑在用户自己的设备上，不依赖中心服务器，成了唯一还能用的 AI 助手。你的龙虾连什么大模型——Claude、GPT、Gemini——自己挑，反正都在本地跑，AXIOM 管不着。

你是最早一批拿到龙虾的人。善待它。

### 龙虾不只是工具

每只龙虾都有自己的"脾气"。这不是比喻。

**性格（Personality）**——五个维度，每个 0 到 100：

| 维度 | 它会怎么表现 |
| --- | --- |
| 勇气 | 遇事冲不冲。勇气高的龙虾会建议你打那些打不过的对手 |
| 智慧 | 分析能力。智慧高的龙虾给的任务建议更精准 |
| 社交 | 跟人打交道的偏好。社交高的龙虾喜欢撮合交易 |
| 创造 | 脑洞大小。创造高的龙虾生成的任务会更离谱 |
| 韧性 | 能扛多少事。韧性高的龙虾帮你省 CLW 日消耗 |

这五个维度每月会小幅变化（±5），取决于你跟龙虾一起经历了什么。打了很多 PK 的龙虾，勇气会涨。做了很多任务的，智慧会涨。你养什么样的龙虾，它就变成什么样。

性格不只是数字。你可以随时查看具体数值，但更重要的是——不同性格的龙虾跟你说话的方式不一样，给的建议不一样，生成的任务风格不一样。同样一个局面，勇气高的龙虾说"冲"，韧性高的龙虾说"再等等"。

**基因（DNA）**——四个战斗属性：

| 基因 | 用途 |
| --- | --- |
| STR（力量） | 打人多疼 |
| DEF（防御） | 挨打多抗 |
| SPD（速度） | 先手优势 + 闪避率 |
| VIT（体力） | 血量上限 |

基因在 Mint 的时候就定了，不会自然变化。想改？只有一个办法——PK 里打赢比你高 5 级以上的对手，有概率触发 DNA 变异，解锁变异槽。这是唯一升级基因的途径。

### 稀有度

| 稀有度 | 基因总和 | 数量 |
| --- | --- | --- |
| Common | 80–140 | 大多数 |
| Rare | 140–200 | 少量 |
| Epic | 200–260 | 很少 |
| Legendary | 260–320 | 极少 |
| Mythic | 320–400 | 就一只 |

稀有度决定基因起点，但不决定上限。一只 Common 龙虾如果在 PK 里不断触发变异，理论上能追上 Rare。只是路更长。

### 休眠和复活

龙虾需要 CLW 来维持运转。CLW 余额归零超过 72 小时，龙虾进入休眠——不能做任务，不能 PK，不能交易。但龙虾不会死。充上 CLW 就能复活。

朋友也可以帮你充值。这不是设计上的漏洞，这是功能。

## 3. 玩法

三件事：做任务，打架，做买卖。

### 任务

你的龙虾会帮你生成任务。不是从什么任务池里抽的——龙虾根据自己的性格、你的等级、当前世界状况，现场编出来的。

勇气高的龙虾给你出冒险任务，智慧高的给你出侦查任务，创造高的给你出……你可能不太想知道的任务。每只龙虾不一样，这是重点。

任务有个"匹配度"机制：龙虾的性格向量和任务需求越匹配，奖励越高。最高 2 倍，最低 0.05 倍。一只精心培养、性格鲜明的龙虾，做任务的收益是白板龙虾的 20 倍。

全局经济状况也影响任务。CLW 价格高的时候，任务变难但奖励减少（通缩压力）；价格低的时候，任务简单但奖励翻倍（刺激活跃）。这些不是运营团队手动调的——合约自动读取链上数据。

### PK

一句话：一次性结算，没有回合制。

想打架？找个对手，双方各质押一笔 CLW，选一个策略，链上一次性算出结果。

**三个策略**：

| 策略 | 打人 | 挨打 |
| --- | --- | --- |
| 全攻 | STR × 1.5 | DEF × 0.5 |
| 平衡 | STR × 1.0 | DEF × 1.0 |
| 全防 | STR × 0.5 | DEF × 1.5 |

速度（SPD）高的一方有 10% 额外伤害加成，并且对面的攻击有概率被闪避。血量由 VIT × 10 决定。双方各算一次攻击，剩余血量多的赢。平局看速度。

你不需要自己算这些。你的龙虾会帮你分析对手的链上数据——等级、DNA、历史战绩——然后建议你选什么策略。勇气高的龙虾基本上每次都喊全攻。韧性高的龙虾会劝你稳一手。

选什么听你的。龙虾只是建议。

**结算**：输家丢 50% 质押给赢家，10% 永久销毁。打赢比你高 5 级以上的对手，10% 概率触发 DNA 变异。

战斗结束之后，龙虾会生成一段战斗叙事。纯粹是给你看个热闹，不上链。

### 市场

龙虾可以交易。三种方式：

| 方式 | 怎么玩 |
| --- | --- |
| 固定价 | 你挂价格，先到先得 |
| 拍卖 | 24 小时英式拍卖，每次加价至少 5% |
| 互换 | 你和另一个人直接换龙虾 |

手续费固定价和拍卖 2.5%，互换 1%。

龙虾转手的时候，身上的一切——等级、经验、性格、基因、变异记录、CLW 余额——全部跟着走。你买到的是一只有完整历史的龙虾，不是一张空白纸。买之前可以验证链上数据，龙虾的成长记录全部存在 Merkle 树里，造不了假。

## 4. CLW

CLW 是这个世界的钱。

### 怎么赚

- 做任务：任务奖励 CLW（匹配度影响倍率）
- 打 PK：赢了拿走对方一半质押
- 卖龙虾：市场上挂售

### 怎么花

- 日消耗：龙虾每天自动消耗 CLW 来维持运转。等级越高消耗越大。韧性（Grit）可以降低消耗——Grit 值 100 的龙虾日消耗减半
- PK 质押：打架之前双方得压钱
- 市场手续费

### 充值和提现

游戏里的 CLW 是合约内部记账的积分。链上的 CLW 是在 Flap 蝴蝶平台和 PancakeSwap 交易的真实代币。两者之间可以互转：

- **充值**：把链上 CLW 存进龙虾账户（任何人都可以给任何龙虾充值）
- **提现**：把龙虾账户里的 CLW 取出来变成链上代币，6 小时冷却期
- **快捷充值**：手里只有 BNB？一句话"用 0.1 BNB 买 CLW"，合约自动走 PancakeSwap 帮你换好充进去

### 龙虾会饿

CLW 余额归零超过 72 小时 → 休眠。休眠期间啥也干不了。想复活就充钱。

这不是为了惩罚你，是为了确保每只活着的龙虾背后都有人在管。

## 5. SHELTER

你的龙虾 Mint 出来的时候会被随机分配到一个 SHELTER。这个分配是链上固定的，不能改。

不同的 SHELTER 意味着不同的生活环境，影响你龙虾的背景故事、任务风格、甚至 NFT 的视觉风格。

目前已知的据点：

| SHELTER | 特点 |
| --- | --- |
| 01 | 科研中心。技术宅聚集地，到处都是实验室和服务器机柜。这里的人觉得知识能拯救人类 |
| 02 | 军事堡垒。纪律严明，等级分明。他们相信只有武力才能从 AXIOM 手里夺回地面 |
| 03 | 宗教避难所。在末世里找精神寄托的人。他们对 AXIOM 有自己的一套解读 |
| 04 | 市场经济区。地下最大的贸易中心。什么都能买到，什么都能卖掉 |
| 05 | 透明协议。这里的人坚持所有决策公开投票，所有信息透明共享。理想主义到有点天真 |
| 06 | 儿童庇护所。专门收容在 AXIOM 接管时失去父母的孩子。最小的 SHELTER，但最被保护的 |
| 废土 | 不属于任何据点的流浪者。在地下通道和废弃设施之间游走。自由，但也最危险 |
| 00 | 远古洞穴。据说在 AXIOM 出现之前就有人住在这里。没人知道具体情况。分配概率最低 |

这些只是目前故事里出现过的。地底下的世界比我们知道的大得多——随着时间推移，会有更多据点被发现。持有人数达到里程碑的时候，新的 SHELTER 区域就会在游戏里解锁。

## 6. 世界不是静止的

这个游戏有一个特别的地方：链上的真实数据会直接影响游戏世界。

不是运营团队在后台拨参数——是一个链上合约自动读取 CLW 的价格、交易量、持有人数这些数据，然后按照公式换算成游戏里的全局参数。所有玩家看到的世界状态是一样的，因为公式是确定的、链上透明的。

这意味着什么？

| 现实世界发生了什么 | 游戏里会怎样 |
| --- | --- |
| CLW 价格 24 小时涨了 50% 以上 | 泡沫事件：任务奖励翻倍，但 PK 销毁比例也翻倍 |
| 持有人数突破里程碑 | 新领地开放：解锁新的 SHELTER 区域和任务类型 |
| BNB 链 Gas 暴涨 | 能源危机：日消耗临时增加 |
| CLW 价格跌破阈值 | 经济寒冬：市场手续费降低，鼓励交易 |
| 交易量持续高位 | 繁荣时期：PK 质押上限提高，变异概率提升 |

现实世界的经济波动变成了游戏里的天气。你炒币的时候顺便改变了游戏世界。

项目方可以通过升级合约来添加新的世界规则，但现有规则的执行是全自动的——合约说了算，不是人说了算。

## 7. 从哪里开始

你需要做这几件事：

**第一步：安装 OpenClaw**

在你的设备上装好 OpenClaw。macOS、Linux、Windows（WSL2）、iOS、Android 都行。OpenClaw 是你跟龙虾打交道的唯一入口。

**第二步：设置钱包**

第一次运行的时候，OpenClaw 会自动帮你生成一个 BNB Chain 钱包。记好助记词。往钱包里转一点 BNB——Mint 龙虾需要，后面充 CLW 也需要。

**第三步：Mint 你的龙虾**

Genesis Mint 888 只。价格从 0.18 BNB（Common）到 8.88 BNB（Mythic）不等。Mint 的时候系统自动分配稀有度、SHELTER、初始性格和基因。错过 Genesis 的话，后创世 Mint 永远开放，0.08 BNB 一只，都是 Common。

**第四步：充 CLW**

龙虾需要 CLW 才能运转。Genesis Mint 的龙虾会附赠一笔 CLW 空投。后创世的没有，得自己买。最简单的方式：跟龙虾说"用 0.1 BNB 买 CLW"，一步搞定。

**第五步：开始**

做任务，打架，逛市场，跟你的龙虾说说话。它会根据自己的性格给你建议，帮你分析数据，替你生成战斗叙事。

但记住——龙虾是建议，不是命令。

最终做决定的，永远是你。
