# 龙虾文明宇宙 — 零预算出图操作手册

**从零到 888 张 NFT · Windows · 不花一分钱**

---

## 总览：你要做什么

```
第一步：用免费AI工具生成「风格锚点」（确定画风长什么样）
第二步：画9层Trait素材（每层是一张透明底PNG）
第三步：用免费工具把9层素材随机组合成888张图
第四步：生成888份JSON元数据
第五步：上传IPFS
```

总耗时：7-14天（一个人）
总花费：0元

## 第一步：生成风格锚点（第1-2天）

目标：用AI生成5-10张样图，确定"你的NFT长什么样"。

### 免费AI工具（选一个）

| 工具 | 免费额度 | 推荐度 | 说明 |
|------|---------|--------|------|
| **Bing Image Creator** | 无限（需微软账号） | ★★★★★ | 基于DALL-E 3，质量最高，完全免费 |
| **Leonardo.ai** | 每天150 tokens | ★★★★ | 质量好，有风格控制 |
| **Stable Diffusion（本地）** | 无限 | ★★★ | 完全免费但需要配置，显卡至少6GB显存 |
| **ChatGPT图片生成** | 免费账号有额度 | ★★★★ | 如果你有ChatGPT账号 |

**推荐用 Bing Image Creator**（bing.com/create），零门槛，不要钱，效果好。

### 操作步骤

1. 打开 bing.com/create
2. 登录微软账号（没有就注册一个，免费）
3. 输入以下提示词生成样图：

```
Bust portrait of a cartoon character in Bored Ape Yacht Club art style.
Thick black outlines, flat color blocks, no gradients, no shading.
A young man with messy hair and a confident smirk, wearing a tactical vest.
On his right shoulder sits a small red cybernetic lobster with glowing eyes
and mechanical claw modifications. Solid teal background.
Square format 1024x1024.
```

4. 生成4张，挑最接近你想要的那张
5. 换不同描述多试几轮（换肤色、换服装、换龙虾颜色、换表情）
6. 直到你找到一张"就是这个感觉"的图——这就是你的**风格锚点**

### 关键提示词公式

```
Bust portrait, Bored Ape Yacht Club cartoon style, thick black outlines,
flat color blocks, no gradients, [角色描述], on shoulder sits a [颜色]
cybernetic lobster with [改造描述], solid [背景色] background, 1024x1024.
```

你只需要换中括号里的内容。

## 第二步：制作Trait素材（第3-8天）

这是最花时间的一步。你需要画出9层素材，每层若干个选项。

### 需要的免费工具

| 工具 | 用途 | 下载地址 |
|------|------|---------|
| **GIMP** | 替代PS的免费图片编辑器 | gimp.org |
| **Krita** | 免费绘画软件（比GIMP更适合画画） | krita.org |

你会PS就直接用PS。不会的话装GIMP或Krita，操作逻辑差不多。

### 素材规格

- 每张素材：1024 × 1024 像素
- 格式：PNG（透明背景）
- 每层的素材位置必须对齐（叠在一起时要能正确拼合）

### 9层素材清单

```
layers/
├── 01_background/     ← 10张（纯色块，不透明）
├── 02_skin/           ← 8张（身体轮廓+肤色，透明底）
├── 03_hair/           ← 15张（发型，透明底）
├── 04_expression/     ← 8张（五官+表情，透明底）
├── 05_outfit/         ← 20张（服装，透明底）
├── 06_accessory/      ← 10张（配饰，透明底）
├── 07_lobster_body/   ← 8张（龙虾身体颜色，透明底）
├── 08_lobster_pose/   ← 6张（龙虾姿态变化，透明底）
└── 09_lobster_mod/    ← 5张（龙虾改造部件，透明底）
```

总共需要画：**90张素材**

### 怎么画（不会画画的人的方法）

**方法A：AI生成 → PS/GIMP抠图 → 拆层**

1. 用Bing Image Creator生成一张完整的角色图
2. 在PS/GIMP里把这张图拆成各个部件：
   - 用魔棒/套索工具把头发单独抠出来 → 存为 hair_01.png
   - 把服装抠出来 → 存为 outfit_01.png
   - 把龙虾抠出来 → 存为 lobster_01.png
3. 重复这个过程，每次换不同的描述生成新图，抠出不同的部件
4. 确保所有部件在画布上的位置一致（肩膀在同一个位置，头在同一个位置）

**方法B：直接在Krita/PS里画**

如果你会一点PS，BAYC的画风其实是最容易模仿的——因为它就是粗线条+色块填充，不需要画阴影和渐变。

1. 先画一个基础的身体轮廓模板（一个，所有素材共用这个轮廓位置）
2. 在模板基础上画不同的发型、服装、配饰
3. 每画完一个就导出为透明底PNG

**方法C：方法A+B混合（推荐）**

1. 用AI生成大量参考图，找到你喜欢的画风
2. 用Krita照着参考图手动描线（BAYC风格的粗线条很好描）
3. 色块用油漆桶直接填

### 对齐技巧

所有素材必须在同一个1024×1024画布上对齐。方法：

1. 在PS/GIMP里创建一个1024×1024的画布
2. 先画一个半透明的"定位参考线"——标出头顶位置、肩膀位置、龙虾位置
3. 每画一个新素材，都在这个参考线上画
4. 导出时隐藏参考线，只导出当前层

### 工作量估算

| 层 | 数量 | 每张耗时（估） | 小计 |
|----|------|---------------|------|
| 背景 | 10 | 2分钟（纯色块） | 20分钟 |
| 肤色/身体 | 8 | 30分钟 | 4小时 |
| 发型 | 15 | 20分钟 | 5小时 |
| 表情 | 8 | 20分钟 | 2.7小时 |
| 服装 | 20 | 30分钟 | 10小时 |
| 配饰 | 10 | 15分钟 | 2.5小时 |
| 龙虾身体 | 8 | 30分钟 | 4小时 |
| 龙虾姿态 | 6 | 20分钟 | 2小时 |
| 龙虾改造 | 5 | 15分钟 | 1.25小时 |
| **总计** | **90张** | | **约31小时** |

分5-6天做完，每天5-6小时。

### 28张命名角色怎么处理

28张命名角色（神话/传说/史诗/稀有）**不走随机组合**。

它们是单独画的完整图。方法：

1. 用v4美术方案里每个角色的提示词，在Bing Image Creator里生成
2. 挑最好的一张
3. 在PS/GIMP里手动调整细节，确保风格跟Trait素材一致
4. 直接导出为完整的1024×1024 PNG

这28张大概需要额外2-3天。

## 第三步：随机组合生成888张图（第9-10天）

### 工具：HashLips Art Engine（免费开源）

这是NFT圈最主流的批量生成工具。免费的。

### 安装步骤（Windows）

**1. 安装 Node.js**

- 去 nodejs.org 下载 LTS 版本（选Windows Installer）
- 双击安装，一路下一步
- 安装完打开命令提示符（CMD），输入 `node -v`，看到版本号就成功了

**2. 下载 HashLips Art Engine**

- 去 github.com/HashLips/hashlips_art_engine
- 点绿色的 "Code" 按钮 → "Download ZIP"
- 解压到你电脑上（比如 D:\hashlips）

**3. 安装依赖**

打开命令提示符，执行：
```
cd D:\hashlips
npm install
```
等它跑完。

### 配置素材

把你画好的素材按这个结构放进 `layers` 文件夹：

│   ├── army#10.png
│   └── darkgray#10.png
├── skin\
│   ├── light#15.png

### 配置生成参数

打开 `D:\hashlips\src\config.js`，找到这几行改一下：

```javascript
const namePrefix = "Claw Civilization";  // NFT名称前缀
const description = "SHELTER resident of the Claw Civilization Universe";  // 描述

const layerConfigurations = [
  {
    growEditionSizeTo: 860,  // 生成860张（普通级）
    layersOrder: [
      { name: "background" },
      { name: "skin" },
      { name: "hair" },
      { name: "expression" },
      { name: "outfit" },
      { name: "accessory" },
      { name: "lobster_body" },
      { name: "lobster_pose" },
      { name: "lobster_mod" },
    ],
  },
];

const format = {
  width: 1024,
  height: 1024,
};
```

### 运行生成

```
cd D:\hashlips
npm run generate
```

等它跑完。860张图会出现在 `D:\hashlips\build\images\` 文件夹里。
对应的JSON元数据在 `D:\hashlips\build\json\` 里。

### 合并命名角色

把你单独画的28张命名角色图片手动命名（比如 `0001.png` 到 `0028.png`），放进同一个文件夹。然后把860张随机生成的编号从 `0029` 开始。

## 第四步：生成元数据（第10天）

HashLips会自动生成JSON元数据。但你需要手动编辑28个命名角色的元数据。

### 元数据格式（OpenSea标准）

```json
{
  "name": "Claw Civilization #0001",
  "description": "在地下出生，在地下长大，从没见过天空。他肩上的龙虾叫问号...",
  "image": "ipfs://你的CID/0001.png",
  "attributes": [
    { "trait_type": "Rarity", "value": "Legend" },
    { "trait_type": "Name", "value": "Truman" },
    { "trait_type": "Department", "value": "None" },
    { "trait_type": "Personality", "value": "Curious" },
    { "trait_type": "Lobster", "value": "Query" },
    { "trait_type": "SHELTER", "value": "SHELTER-01" }
  ]
}
```

命名角色的背景故事写在 `description` 字段里——就用美术方案v4里每个角色的背景故事。

## 第五步：上传IPFS（第11天）

### 工具：Pinata（免费）

1. 去 pinata.cloud 注册账号（免费）
2. 免费额度：1GB存储 + 每月100次API调用
3. 888张1024×1024的PNG大约 300-500MB，免费额度够用

### 上传步骤

1. 登录Pinata
2. 点 "Upload" → "Folder"
3. 上传你的 `images` 文件夹 → 得到一个 CID（类似 `QmXXXXXXX`）
4. 把所有JSON元数据里的 `image` 字段改成 `ipfs://你的CID/文件名.png`
5. 再上传 `json` 文件夹 → 得到另一个 CID
6. 这个JSON文件夹的CID就是你合约里的 `baseURI`

## 第六步：质检（贯穿全程）

### 生成后必须做的检查

1. **随机抽查50张**：看有没有素材错位（头发飘到肩膀外面、龙虾叠到脸上等）
2. **检查唯一性**：HashLips会自动排重，但还是看一眼日志确认没有重复
3. **检查稀有度分布**：确认稀有Trait的出现次数符合你设定的权重
4. **检查命名角色**：28张定制图的质量是门面，仔细看

### 常见问题

| 问题 | 解决方法 |
|------|---------|
| 素材错位 | 回去检查那一层的PNG定位，确保跟参考线对齐 |
| 颜色不统一 | 建一个色板文件，所有素材从色板里吸色 |
| 龙虾跟人物重叠 | 龙虾层的位置要固定在肩膀区域，画的时候避开头发区域 |
| 线条粗细不一 | 在Krita/PS里统一用同一号画笔（建议5-8px粗） |
