## 一、核心策略：用代码替代美术外包

你的项目有一个天然优势：**龙虾是非人类角色**。非人类角色的像素风比人类角色容易做好看 100 倍——不需要处理五官比例、表情细腻度等问题。CryptoPunks 之所以成功，不是因为画得精细，而是因为风格统一、辨识度高。

**核心思路：** 不请画师，用「AI 生成参考 → 像素化 → 代码批量合成」的流水线，一个人搞定全部美术。

## 二、画风定义：赛博废土像素风

### 2.1 风格关键词

| 维度 | 定义 |
|------|------|
| 大类 | Cyber-Dystopian Pixel Art（赛博废土像素） |
| 分辨率 | 64×64 像素（比 CryptoPunks 的 24×24 精细，但仍保持像素质感） |
| 输出尺寸 | 640×640 px（10 倍整数放大，保持锐利边缘） |
| 色调 | 深色底（#0a0a12 级别的暗）+ 霓虹高光（cyan/red/green/purple） |
| 质感 | 金属锈蚀 + 霓虹发光 + 扫描线噪点 |
| 氛围 | SHELTER 地下基地：昏暗、工业管线、闪烁屏幕 |

### 2.2 色彩体系

**身体色板（8 种）：**
- 深红系（#8b1a1a → #ff8a80）：最常见，龙虾本色
- 铁锈系（#5d3a1a → #ee8262）：废土感
- 深海蓝系（#0d3b66 → #90caf9）：冷酷
- 毒液绿系（#1b5e20 → #a5d6a7）：变异体
- 幻影灰系（#1a1a2e → #7575a3）：幽灵感
- 铬银系（#37474f → #b0bec5）：机械改造
- 琥珀橙系（#e65100 → #ffb74d）：烈焰
- 虚空黑系（#0a0a0a → #3a3a4e）：最稀有

**霓虹高光（8 种）：** #00f0ff · #ff2d55 · #39ff14 · #b829ff · #ff6a00 · #ffe400 · #ff69b4 · #4169e1

**背景色板（6 种）：** 对应不同 SHELTER 环境——暗蓝基地、锈红废土、暗绿丛林、荒漠黄昏、实验室蓝、虚空

### 2.3 为什么这个画风「牛逼」

1. **辨识度极高**：市面上龙虾主题 NFT 几乎为零，赛博废土+龙虾的组合独一无二
2. **像素风有文化认同**：CryptoPunks 验证了像素风的市场价值，在 Web3 社区是"懂行"的象征
3. **深色霓虹=赛博朋克美学**：这个色系在社交媒体上传播效果最好（对比度高、头像显眼）
4. **可进化**：你的 NFT 是动态的（转职、升级），像素风最容易做元数据更新后的视觉变化

## 三、Trait（特征）分层设计

### 3.1 分层架构

从底到顶共 8 层，每层独立绘制后叠加合成：

| 层级 | 名称 | 数量 | 说明 |
|------|------|------|------|
| L1 | 背景 Background | 6 | SHELTER/废土/实验室等场景 |
| L2 | 身体 Body | 8 | 龙虾主体颜色（4 色阶渐变） |
| L3 | 服装 Outfit | 15 | 5 部门 × 3 款式 |
| L4 | 钳子 Claws | 8 | 机械/能量/刃式/黑客等 |
| L5 | 头饰 Headgear | 10 | 头盔/护目镜/天线/全息/王冠等 |
| L6 | 眼睛 Eyes | 6 | 发光眼/扫描器/独眼罩等 |
| L7 | 嘴部 Mouth | 4 | 咧嘴/坚毅/獠牙/坏笑 |
| L8 | 特效 Effects | 5 | 无/微光/粒子/光环/能量翅膀 |

**理论组合数：** 6 × 8 × 15 × 8 × 10 × 6 × 4 × 5 = **5,760,000 种**
远超 888 只的首批供应量，保证每只独一无二。

### 3.2 Trait 与游戏属性关联

| Trait | 关联机制 |
|-------|----------|
| 身体颜色 | 稀有度分布不均：虚空黑仅出现在传说级 |
| 服装 | 直接绑定部门类型（战斗=红色装甲，情报=暗紫斗篷等） |
| 钳子 | 影响视觉战斗风格，与 ATK 属性呈正相关 |
| 头饰 | 体现科技水平，王冠仅限传说 |
| 眼睛 | 对应性格系统（扫描器=谨慎，独眼罩=激进等） |
| 特效 | 仅史诗和传说出现，是视觉稀有度的直接体现 |

### 3.3 Trait 稀有度概率表

**身体颜色稀有度：**
- 深红/铁锈/深海蓝：各 18%（常见）
- 毒液绿/幻影灰/铬银：各 12%（较少）
- 琥珀橙：6%（少见）
- 虚空黑：2%（极稀有，仅传说级）

**特效稀有度：**
- 无特效：70%（全部普通级）
- 微光：12%（部分稀有级）
- 粒子环绕：10%（稀有+史诗）
- 光环：6%（史诗级）
- 能量翅膀：2%（传说级专属）

## 四、零成本创作流水线

### 4.1 方案 A：纯代码生成（推荐，$0 成本）

这是最低成本、最可控的方案。上面的 Demo 已经展示了纯代码画龙虾的效果。

**工作流：**

```
1. 用 JavaScript/Python 定义每个 Trait 层的像素绘制函数
2. 每个函数绘制一个 64×64 的 PNG 透明层
3. 用 HashLips Art Engine 或自写脚本随机组合各层
4. 输出 888 个唯一 PNG + JSON 元数据
5. 上传 IPFS
```

**工具：**
- 绘制引擎：JavaScript Canvas API（本 Demo 已包含完整代码）
- 批量合成：HashLips Art Engine（GitHub 开源免费）
- 元数据生成：Python 脚本（自己写 50 行代码搞定）
- IPFS：Pinata 免费额度 1GB（888 张 640×640 PNG 约 200MB）

**优势：**
- 完全免费
- 100% 可控，随时调整
- 代码即资产，可以开源展示技术实力
- 与"无后端架构"的项目理念完美契合

**时间成本：** 1 人约 3-5 天完成全部 Trait 绘制 + 批量生成

### 4.2 方案 B：AI 辅助像素化（$0-20 成本）

如果你想要更精细的画面效果：

**工作流：**

```
1. 用 Midjourney/DALL-E 生成龙虾参考图
   Prompt: "cyberpunk lobster warrior, dystopian shelter, neon glow, dark atmosphere, character design, front view"
2. 用 Pixel It（免费开源）或 Aseprite（$20 一次性）将 AI 图像素化为 64×64
3. 手动调整/清理像素，确保一致性
4. 拆分各 Trait 层的像素素材
5. 用 HashLips 批量合成
```

**优势：**
- AI 生成的参考图视觉冲击力更强
- 最终像素化后仍保持手工质感
- 有更多细节和层次

**劣势：**
- 需要手动拆层，工作量比方案 A 大
- AI 图的一致性需要人工把控

**时间成本：** 1 人约 7-10 天

### 4.3 方案 C：混合方案（最佳平衡）

**推荐的实际操作：**

```
1. 先用方案 A 的代码生成基础版本（3-5 天）
2. 用基础版本立即开始社区预热和白名单征集
3. 同时用方案 B 迭代优化画面质量
4. 在正式 Mint 前用优化后的版本替换
```

这样你可以最快速度启动项目，同时持续提升美术质量。

### 7.2 转职视觉设计（10 个方向）

每个转职方向需要设计：
1. 新服装层（替换 L3）
2. 新专属特效（替换 L8）
3. 职业标识（在 NFT 名称和元数据中体现）

| 转职 | 视觉主色 | 专属特效 |

| 废土拾荒者 | 土黄+锈色 | 探测雷达 |
| 机械工匠 | 银灰+电弧 | 工具悬浮 |
| 潜行者 | 暗紫+透明 | 隐匿粒子 |
| 数据分析师 | 青色+数据流 | 全息数据屏 |
| 黑市商人 | 金色+暗红 | 金币光晕 |
| 物资调度员 | 橙色+褐色 | 传输光线 |

### 7.3 实现方式

元数据更新 = IPFS 指向新图片 URI。当玩家转职时：
1. 链上合约记录新职业
2. 触发后端脚本（或 Skill）根据新属性重新合成图片
3. 上传新 PNG 到 IPFS
4. 更新 NFT tokenURI
